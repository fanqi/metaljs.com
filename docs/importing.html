导入第三方的组件

比起以前，现在来说有更多的类库和框架可用于前端开发。在单个项目中涉及到5个甚至更多的类库都不是罕见的。但是要保持跟踪这些类库并确保他们是最新的确实非常费精力的。要解决这个问题，我们可以使用 NPM，一个能够管理您所有应用依赖的包管理器。

在这个指南中，您将学会怎样准备和运行 NPM。您将通过安装 NPM 命令行工具，然后继续学习可用于管理 metal 组件相关的各种命令。

让我们开始吧!

安装 Node.js/NPM

如果你还没有安装好的 Node.js 或者 npm，请先去 Node.js 网站下载适合您系统的 Node.js 副本。npm 程序已经包括在 Node.js 的安装中了。

现在您已经安装了 NPM，我们可以看看可用于管理包的命令了。

查找组件

查找 NPM 包有两种不同的方法。使用在线的组件目录或者使用命令行工具。

要在命令行中搜索包您能使用搜索命令。搜索查询示例如下：

npm search <query>

例如，要搜索所有包含单词 'metal' 的包，命令如下：

npm search metal

这个命令将返回一整串结果，包含了每个匹配的组件信息，所以你可以选择你想要的那个。

安装组件

要安装一个新的 NPM 包到您的项目，您能使用安装命令。需要传入您期望安装的包名。 

npm install <package>

例如，我们要安装 metal-position 组件。

npm install metal-position

已经安装的包将自动放在 node_modules 目录中，这是由 bower 程序执行自动创建的。

  • node_modules
    • metal
    • metal-position

随着代码已经可用了，让我们创建一个 main.js 文件来导入 metal-position 模块。注意我们将用 alias 来轻松地导入 npm 文件。

这意味着您现在可能调用那个模块的任何方法，在例子中我们来获取 viewport 的高度。

Metal 组件是用 ES6 编写的（又叫做 ECMAScript 2015），因此您也能像我们在示例中做的那样在您的代码中使用 ES6。由于 ES6 还没有完全被浏览器实现，因此在网站上使用 Metal 还需要一个构建过程。
